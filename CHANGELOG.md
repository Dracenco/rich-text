# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [Unreleased]
### Fixed
- Fixed issue that causes syntax error on IE11, due to the use of ES6 features.

## [0.6.0] - 2019-06-06
### Added
- Support for lists.
- Support heading levels in css.

## [0.5.0] - 2019-06-03

### Added

- Support for table.
- Support for `target` attribute in `<a>` tags.

## [0.4.0] - 2019-05-27
### Added
- i18n content edition support through CMS.

## [0.3.0] - 2019-05-21
### Added
- Allow `iframe` HTML tag.

## [0.2.0] - 2019-04-30
### Added
- Added `htmlId` prop.

## [0.1.5] - 2019-04-25

## [0.1.4] - 2019-04-22
